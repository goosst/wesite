Intro

Use MQTT to send status of alarm to Home Assistant

Troubleshooting

The connection of a Wemos D1 with Arduino IDE seems to be junk. We need to use esptool instead pip3 install esptool

structure to use: esptool --port "{serial.port}" --baud {upload.speed} write_flash 0x00000 "{build.path}/{build.project_name}.bin
